# Greeter
A simple greeter program written in V

# Installation
```
> git clone https://github.com/vanillaiice/greeter
> cd greeter
> v .
```

# Usage
```
> ./greeter
// output:
// What is your name: Jotaro Kujo Joestar
// Hello, Jotaro Kujo Joestar !
```
